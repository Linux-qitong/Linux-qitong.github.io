npm安装（ 新手向）

2023年03月19日
软件
留言

具体安装请看下文。

sudo身份运行下面的命令，下载执行NodeSource安装脚本：前面加sudo空格。

curl -sL https://deb.nodesource.com/setup_18.X

这个根据版本更新变化，比如出19了，改成19.X，20.X等。(如果你需要其他版本，比如旧版本也可以修改。)

这个脚本会添加 NodeSource 的签名到系统中，创建一个apt源文件，安装必备软件包，并刷新apt 缓存。

NodeSource源被启用后，输入命令:

sudo apt install nodejs

nodejs包含node和npm二进制包。

验证是否正确安装

验证node: node –version

验证npm: npm –version

需要从npm编译本地扩展，安装开发工具。

sudo apt build-essential

以上就是这些，感谢大家支持。
